# Actors

| Actor | Description |
|-------|-------------|
| Customer | End user initiating return/refund/claim. |
| Claimlane Portal | Guided interface enforcing rules. |
| WooCommerce System | Stores orders, refunds, replacements. |
| Shipping Carrier API | FedEx/EasyPost/freight services. |
| CS Agents | Review claims, override docs, manual refunds. |
| Vendors/Warehouses | Receive returns and approve pickups. |
| Retail/POS Systems | Manual store returns and guidance. |
| Marketing Team | Provides approved portal copy. |
| QA Testers | Validate behaviour and rules. |
